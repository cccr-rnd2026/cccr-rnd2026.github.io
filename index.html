클라우드 네이티브기반 GPU 오케스트레이션: AI Factory

AI워크로드 특성을 고려하여 GPU 클러스터 자원의 효율적 운영·관리를 제공하는 GPU 오케스트레이션 기술 개발

Guide Documentation (KOR) 📖GitHub 🚀

About Project

다양한 특성을 갖는 AI 학습 및 추론 워크로드 구동 환경에서 GPU 자원의 동적 할당·스케줄링·자동화 기술을 적용하여 자원 효율성과 운영 안정성을 동시에 확보하고, GPU 클러스터의 활용률을 극대화하는 GPU 오케스트레이션 기반 AI Factory SW 개발

Technology

핵심기술 #1

AI 워크로드 특성을 고려한 클러스터 내 GPU 자원 풀링 및 분산 런타임 기술

GPU 클러스터의 논리적 자원 풀 통합 관리 및 워크로드(학습&추론 특화) 분산 실행 런타임 엔진 기술 개발💡 세계최초
  • ―물리적으로 이산된 이종 GPU 자원의 하나의 논리적 로 추상화 및 동종 연속자원 제공기술
    → 자원 호환성 보장 및 학습, 추론 성능 향상
  • ―AI 워크로드 유형별로 학습 속도 최대화, 추론 처리량 증가를 위한 이종 GPU 자원 인지 분산 런타임 엔진
    → AI 워크로드 분산 실행 성능 보장
핵심기술 #2

이종 GPU 환경에서의 성능 보장 DRF 공정성 스케줄링을 위한 동적 자원 선점·할당 기술

Fractional GPU 클러스터 환경에서 학습·추론 작업을 통합 지원하는 공정한(상대처리율) 스케줄링 기술 개발💡 세계최초
  • ―AI워크로드간 상대 처리율 격차 완화(공정성 보장) + 학습·추론 적합 배치 스케줄링
    → AI 워크로드 성능 보장 + 자원 활용 효율 향상
핵심기술 #3

GPU 클러스터의 효율적 운영을 위한 다차원 프로파일링 및 모니터링 기술

워크로드(모델) 특성과 인프라(GPU) 상태를 시계열 다차원으로 통합 분석하는 다차원 연계 분석 엔진 개발💡 세계최초
  • ―AI 워크로드 실행 ↔ GPU 자원 사용까지 클러스터 관점에서 다차원 분석을 통해 정확한 연산 자원 병목 지점 및 인과관계 진단
핵심기술 #4

멀티 테넌트 기반 워크로드 최적 성능 수행 보장 기술

자원 격리·동적 분배 및 Fair-share + 프로파일기반 간섭 인지형 워크로드 최적 성능 보장 기술🚀 세계동급
  • ―Guaranteed Quota + 동적 Resizing-Quota 기반 논리적 자원 격리
    → 성능 간섭 최소화
  • ―실행 중 자원 사용률·경합 상태 반영 분석 데이터 기반의 동적 자원 재균형 및 Reclaim 제어 기술
    → 테넌트별 최적 성능 제공
핵심기술 #5

AI 워크로드의 상태 기반 전주기 관리·자동화를 지원하는 DevOps 기술

개발-학습-배포-운영 전주기를 자동화하고, 서빙 성능 보장을 위한 자동 재학습을 지원하는 DevOps 기술🚀 세계 동급
  • ―AI 워크로드 템플릿 생성 시 전주기 상태와 이력 기록 + 모델 배포 후 요청 패턴 & 실시간 분석
    → 배포 및 동시성 자동 조정
  • ―기존 코드 중심 CI/CD 구조를 넘은 AI 워크로드 상태·서빙 SLO를 연계한 전주기 재실행·재구성 제어 체계 개발
핵심기술 #6

운영 관리 성숙도 모델 Level-4 수준 GPU 클러스터 AIOps 자동화 기술

이기종 GPU 클러스터의 설명 가능한 AI 모델 기반 AI SW 스택 자율 구성 및 자율 복구 기술 개발💡 세계최초
  • ―GPU 특화 사용량 예측 + 자원 최적화 에이전트 + 스케일 제어·SW 스택 자동 구성
    → 동적 자원 최적화, AI 서비스 확장, 운영 효율성 극대화
  • ―MELT(Metric, Events, Logs, Traces) 멀티모달 탐지모델 + 인과그래프 기반 원인 분석 + 지식 그래프 기반 자율 복구
    → 평균 복구 시간 단축
핵심기술 #7

멀티 테넌트기반 GPU 오케스트레이션 AI Factory 운영 관리 플랫폼

AI Factory 플랫폼의 AI SW 스택을 수직 통합 제어하는 Control Plane 기술 개발🚀 세계동급
  • ―Federated Control Plane 기반 컴포넌트에서 AIOps까지의 전주기 통합 관리 구조 제공
    → AI Factory Control Plane 기능 통합
  • ―계정·자원·워크스페이스정책기반 인증 및 접근·변경 기록 감사 로깅
    → 운영 안정성 및 통합 거버넌스 확보
  • ―GPU 자원 추상화 + 오케스트레이션 + DevOps + AIOps = 단일 Vertical AI SW 스택
    → AI Factory SW 운영체계 확보(표준연계)
